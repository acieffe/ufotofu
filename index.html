UFO Tofu: A Coding Bootcamp Project

UFO Tofu sounds like just another extra-terrestrial bean curd, but it also a palindrome. It can be read both forwards and bakwards the same. This application checks if the user input is a palindrome or not and displays a message of the outcome.
